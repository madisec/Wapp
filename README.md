# Wapp
A small Wappalyzer tool for recon

## Install
```bash
git clone https://github.com/madisec/Wapp.git
cd Wapp
go build main.go
mv main.go wapp
chmod +x wapp
sudo mv wapp  /usr/bin/
```
## How to use
```bash
wapp [ URL ]
```
